Gaming & Dev: How Video Games Are Created – Website Planning Document

1. Site Name

Site Name: Gaming & Dev: How Video Games Are Created

This name was chosen because it clearly represents the website’s focus: teaching beginners how video games are made. It combines both interests — gaming and programming — in an accessible and attractive way for new learners.

2. Site Purpose

The purpose of this website is to guide beginners in understanding how video games are created. It will provide:

  • Clear explanations of the game development process.
  • Introductions to popular game engines and their uses.
  • Beginner-friendly programming concepts related to games.
  • Step-by-step starting guides and practice project ideas.
  • Resources to help users continue learning game development.

3. Scenarios

  • How do I choose the right game engine for my first game project?
  • What basic programming concepts should I learn to start making games?

4. Color Schema

Neon Red (#FF3C28): Used for headings, accents, and interactive elements to convey creativity and technology.

Black (#000000): Used for body text to improve readability and contrast.

Neon Blue (#0AB9E6): Used for card backgrounds and section separators for a clean layout.

5. Typography

Oswald: Used for headings and titles, giving a bold, structured visual style.

Roboto: Used for body paragraphs and lists to improve readability and maintain a modern look.
